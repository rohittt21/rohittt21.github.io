# Rohit Loharkar — AI & Software Engineer Portfolio

Welcome to the source code of my personal developer portfolio! 
You can view the live website here: [https://rohittt21.github.io](https://rohittt21.github.io)

## 🚀 Overview

This is a modern, fully responsive, and highly animated single-page developer portfolio designed to showcase my journey as an AI/ML Engineer and Data Sciences M.Tech candidate. It highlights my technical arsenal, professional timeline, projects, and research publications.

## ✨ Key Features

- **Modern Glassmorphism UI**: A sleek, translucent design system with frosted glass effects and dynamic glowing accents.
- **Advanced CSS Animations**: Custom `@keyframes` for floating particles, rotating "solar system" orbital rings, and staggering reveal animations on scroll.
- **Custom Cursor & HUD**: A dynamic cursor with snap effects and a persistent navigation Heads-Up Display (HUD).
- **Responsive Layout**: Fluidly adapts from desktop to mobile screens using CSS Grid and Flexbox.

## 🛠️ Technology Stack

- **HTML5**: Semantic and accessible structure.
- **CSS3 (Vanilla)**: Advanced styling, CSS variables for theming, Grid/Flexbox layouts, and hardware-accelerated animations. No external frameworks used.
- **JavaScript (Vanilla)**: Intersection Observers for scroll animations, cursor tracking, dynamic HUD updates, and particle canvas rendering.

## 📂 Project Structure

- `index.html` — The main entry point and structure of the portfolio.
- `css/styles.css` — All styling, theming, responsive queries, and keyframe animations.
- `js/scripts.js` — Logic for custom cursor, scroll reveals, particle canvas, and navigation tracking.
- `images/` — Assets and graphics (logos, project previews).
- `Resume Rohit Loharkar.pdf` — Downloadable CV.
